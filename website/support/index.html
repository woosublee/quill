Open GitHub Issues

Do not post sensitive information

GitHub issues are public. Do not include any of the following in public issues:

  • OAuth credentials or API keys.
  • Private Google Calendar event details.
  • Sensitive transcripts, audio recordings, screenshots, or selected text.
  • Personal information that should not be public.

Google Calendar connection help

Google Calendar integration is optional. In Quill Settings, connect Google Calendar, then select the calendars Quill may use for note title suggestions. No calendars are selected by default.

To stop using the integration, disconnect Google Calendar in Quill Settings. This removes Quill's locally stored Google Calendar OAuth token and clears the selected calendars.

You can also revoke Quill's Google access from your Google Account permissions page.
