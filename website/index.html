Quick Start

1Download the app using the button at the top of this page or click here.
2Get a free Groq API key at groq.com.
3Hold Fn to talk, or tap Command-Fn to start and stop dictation, and have whatever you say pasted into the current text field.

Why Groq?

Groq provides extremely fast transcription and LLM responses (<1s) for a smooth experience. Local models currently introduce too much latency (5-10s) and can impact battery life.
