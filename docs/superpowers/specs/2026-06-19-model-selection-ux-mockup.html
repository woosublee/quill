로컬 모델 상태 흐름

로컬 모델은 고른다고 바로 쓰이지 않습니다. 설치 상태에 따라 행 아래 영역이 바뀝니다 — 기존 전사 모델 다운로드 UI(ModelRowView / TranscriptionModel.download)를 모든 로컬 모델로 일반화한 것입니다.

  • 미설치모델명 · 예상 용량 + [다운로드] 버튼. 선택은 됐지만 받기 전엔 실제 사용 보류.
  • 다운로드 중진행률 바 + %. (취소 가능) 다운로드 동안에도 다른 기능은 계속 설정 가능.
  • 설치됨바로 사용. 로컬 LLM은 BYO 안내(Phase 2) 또는 무설치(Phase 3).
  • 내장Apple Speech 등은 다운로드 불필요.
  • 제약로컬 컨텍스트는 "텍스트만 · 화면 분석 불가" 경고.

구현 메모
