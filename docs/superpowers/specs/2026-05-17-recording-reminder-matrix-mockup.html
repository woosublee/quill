Final matrix · recording / reminder / processing / default reminder

최종 후보 기준입니다. 녹음 종료 직후에는 알림 포함 panel 안에서 recording overlay만 processing indicator로 바뀐 상태를 유지합니다. 처리가 완전히 끝나면 녹음 상태가 아니므로 기본 meeting reminder panel로 자연스럽게 전환합니다.

상태 전환 원칙

1. 녹음 오버레이
현재 recording layout 그대로
2. 알림 포함
기존 녹음 overlay 유지 + reminder 정보 추가
3. 녹음 종료 / 처리 중
같은 알림 포함 panel 유지, 녹음 overlay만 processing으로 변경
4. 완전 종료
기본 meeting reminder panel로 전환

Start 동작과 외부 녹음 시작 동작

알림에서 Start를 누른 경우
알림 panel을 먼저 닫고, 그 다음 일반 녹음 overlay만 내려옵니다. 이때 reminder 정보는 녹음 overlay에 합쳐지지 않습니다.
Start를 누르지 않고 녹음이 시작된 경우
기존 reminder가 아직 떠 있으면, 새 recording overlay 상태를 기준으로 알림 포함 형태로 전환됩니다. 즉 녹음 중 정보용 reminder로 계속 표시됩니다.
